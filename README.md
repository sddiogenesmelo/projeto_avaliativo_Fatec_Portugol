# Projeto Avaliativo ADS Fatec-Itatiba

🍕 Pizzaria Morte Lenta - Sistema de Pedidos (Portugol)
Este projeto foi desenvolvido como parte da Avaliação P1 da disciplina de Algoritmos e Lógica de Programação do curso de Analise e Desenvolvimento de Sistemas na FATEC Itatiba.

O objetivo foi criar um sistema de automação para uma pizzaria, abrangendo desde o login/cadastro do usuário até a geração de um cupom não fiscal com lógica de descontos.

🚀 Funcionalidades
Sistema de Login: Validação de acesso com senha de 8 dígitos.

Cadastro de Clientes: Fluxo para novos usuários com coleta de dados (nome, telefone, endereço).

Cardápio Interativo: Escolha de múltiplos sabores com soma automática de valores.

Lógica de Negócio: Sistema que aplica automaticamente 10% de desconto para pedidos acima de R$ 100,00.

Emissão de Ticket: Geração de um resumo detalhado do pedido (Cupom Não Fiscal).

🛠️ Tecnologias e Ferramentas
Linguagem: Portugol (Portugol Studio).

Bibliotecas Utilizadas:

Texto: Para validação da quantidade de caracteres da senha.

Matematica: Para o arredondamento de valores monetários e cálculos de porcentagem.

🧠 Desafios e Aprendizados
Durante o desenvolvimento, enfrentei e superei os seguintes desafios técnicos:

Modularização: O código foi estruturado em funções específicas (pedido, cadastrarUsuario, gerandoRelatorio) para facilitar a manutenção e a legibilidade.

Escopo de Variáveis: Aprendi a trabalhar com variáveis globais para permitir que diferentes funções compartilhassem dados do cliente e do carrinho.

Tratamento de Dados: Implementação de inicialização de variáveis para evitar erros de memória e garantir que o relatório não fosse gerado sem dados prévios.

Pesquisa e Mentoria: O projeto envolveu leitura de documentação oficial e busca por mentoria externa para entender conceitos de arquitetura e lógica de controle (flags booleanas).
